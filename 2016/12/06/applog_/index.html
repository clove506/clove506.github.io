优护助手3.0埋点文档

key名

请求埋点格式：_ request app _ [requestName]
requestName与H5保持一致

页面埋点格式：_ page app _ [pageName]

value格式

埋点格式：

@”hospitalId” : 医院id
@”hospital” : 医院名
@”departmentId” : 科室id
@”department” : 科室名
@”nurseId” : 用户Id
@”phone” : 手机号
@”currentVersion” : 当前版本号
@”env” : 当前环境代号如下

YHJModeDebug = 0,    // 对内测试
YHJModeDev = 1,      // 对外测试
YHJModeRelease = 2,  // 正式
YHJModeStaging = 3   // 灰度
