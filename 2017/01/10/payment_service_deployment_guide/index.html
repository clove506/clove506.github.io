部署方法

  • cd /opt/zhushou
  • git clone -o origin git@github.com:youhujia/midas.git
  • cd midas
  • ./bin/start.production
  • 或者: CONFIG_URL=http://10.0.0.6:5001/ IP=10.0.0.6 PORT=6011 MPORT=6012 ENV=production gradle go
  • docker ps 确认运行状况
