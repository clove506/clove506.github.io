订单详情

GET /api/nurses/order/v1/{orderId}/detail/

描述

点击订单内容展示区或者内容服务中，进入订单详情页。
详情页包括：待接单，待服务，服务中,服务结束，用户申请退款。
服务中状态包括：1.用户申请退款被拒绝，进入纠纷，2.服务人员服务中申请进入纠纷。
